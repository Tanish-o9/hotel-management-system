# 🏨 Hotel Management System

A modern, responsive, and full-stack Hotel Management System that simplifies hotel discovery, booking, and management. The application provides an intuitive platform where users can search for hotels, explore room details, compare prices, and make reservations seamlessly. Administrators can efficiently manage hotel listings, room availability, bookings, and customer information through a dedicated dashboard.

## ✨ Features

* 🔐 Secure User Authentication (Sign Up/Login)
* 🏨 Browse and Search Hotels
* 🔍 Filter Hotels by Location, Price, and Amenities
* 🛏️ View Detailed Hotel & Room Information
* 📅 Online Room Booking and Reservation Management
* 📖 Booking History for Users
* 👨‍💼 Admin Dashboard for Hotel & Booking Management
* 📱 Fully Responsive Design
* ⚡ Fast and User-Friendly Interface
* 🗄️ Secure Database Integration
